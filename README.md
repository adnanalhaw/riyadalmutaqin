# رياض المتقين — riyadalmutaqin.com

موقع قناة «رياض المتقين» الدعوية: القرآن والسنة بفهم سلف الأمة.
تصميم أخضر ليلي + ذهبي (هوية 2026 المعتمدة).

## الصفحات

| الملف | الوظيفة |
|-------|---------|
| `index.html` | الصفحة الرئيسية: بطاقات المنصّات، الآية، شارك/التدريس/الخصوصية/الدعم، مبدّل اللغة 🌐 |
| `login.html` | تسجيل الدخول بأربعة أدوار: متعلم · معلم · مدير الموقع · مدير النظام |
| `dashboard.html` | لوحة التحكم — تتشكّل حسب دور الحساب |
| `studio.js` | 🎨 استوديو البوستات: تصميم واحد → 8 مقاسات (تيك توك، ريلز، ستوري، منشور إنستغرام، واتساب، إكس، فيسبوك، مصغّرة يوتيوب) مع مراعاة المنطقة الآمنة لكل منصّة |
| `auth.js` | الحسابات والصلاحيات (بصمات SHA-256 فقط — لا كلمات مرور في المستودع أبداً) |
| `app.js` | روابط المنصّات (`SITE.links`) والمنطق المشترك — عدّل الروابط هنا |
| `styles.css` | نظام التصميم |

## الصلاحيات

| الدور | يرى |
|-------|-----|
| متعلم | بطاقات قنوات التعلم |
| معلم | استوديو البوستات |
| مدير الموقع | الاستوديو + روابط المنصّات |
| مدير النظام | كل شيء + إدارة الحسابات ومولّد بصمات كلمات المرور |

## النشر (Cloudflare)

```bash
npx wrangler deploy
```
أو اربط المستودع بـ Cloudflare Pages فيُنشر تلقائياً مع كل دفعة إلى main.

## ملاحظات

- **الشعار:** ضع ملف `logo.png` في جذر المستودع (يظهر بديل مؤقت إن غاب).
- **تغيير كلمة مرور:** لوحة مدير النظام → «مولّد البصمة» → انسخ البصمة الجديدة
  إلى `auth.js` وادفع التعديل. كلمات المرور نفسها لا تُخزَّن في أي مكان.
- **حدود الأمان بصدق:** التحقّق يجري في المتصفح (موقع ثابت بلا خادم). يكفي لأدوات
  لوحة لا تحمل بيانات حسّاسة؛ للترقية لاحقاً: Cloudflare Worker + secrets.
